# osTicket - Ticket Lifecycle: Intake Through Resolution

This project covers the ticket lifecycle in osTicket from intake to resolution. The goal is to show how tickets are created by end users, worked by help desk agents, and resolved using different priorities, departments, and SLAs.

## Project Summary
This project walks through the lifecycle of multiple tickets inside osTicket, from creation to final resolution.

**What this project is:**  
A walkthrough of ticket intake, assignment, working the issue, and resolution in osTicket.

**Languages Used:**  
None

**Environments Used:**  
Microsoft Azure Virtual Machine  
Remote Desktop  

**Technologies / Services Used:**  
osTicket  
IIS  
Windows 10 (21H2)
